# SimpleApp

Andorid app for challenge purposes about searching and listing books and visualize book details

# Feature

- Book listing using Compose.
- BookDetail screen with book data.
- Error handling.
- Dependency injection.

### Dependencies

- **Kotlin:** Main programming language.
- **Jetpack Compose:** Declarative UI framework.
- **Hilt:** Dependency injection for easier management of components and testing.
- **Coil:** Image loading in Compose.
- **JUnit & Mockk:** Unit and UI testing.
